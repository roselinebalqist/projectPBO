Program ini merupakan simulasi pengelolaan penumpang bus Trans Koetaradja yang dibuat dalam code Java, yang memungkinkan penambahan, pengurangan, dan pencarian penumpang dengan pembagian kursi sesuai aturan prioritas. Fitur interaktif command line menyediakan menu untuk melihat laporan jumlah penumpang, pendapatan bus, aktivitas harian, serta filter penumpang berdasarkan tipe atau ID, sehingga memudahkan pengelolaan data penumpang secara efektif dan fleksibel.
